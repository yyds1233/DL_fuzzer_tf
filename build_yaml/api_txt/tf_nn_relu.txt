tf.nn.relu

Computes rectified linear: max(features, 0).

tf.nn.relu(
    features,
    name=None
)

Computes the rectified linear unit activation function. Given a tensor features,
this operation computes features * (features > 0), i.e., it returns
max(features, 0) element-wise.

Args:
  features: A Tensor. Must be one of the following types: float32, float64, int32,
    uint8, int16, int8, int64, bfloat16, uint16, half, uint32, uint64, qint8.
    The input tensor. Can have any number of dimensions (any rank).
  name: A name for the operation (optional).

Returns:
  A Tensor. Has the same type and shape as features.

The input features can have any rank (any number of dimensions). The operation
is applied element-wise.

Examples:
  tf.nn.relu([-2., 0., 3.])  =>  [0., 0., 3.]
  tf.nn.relu(tf.constant([[-1, 2], [3, -4]], dtype=tf.float32))
    =>  [[0., 2.], [3., 0.]]
