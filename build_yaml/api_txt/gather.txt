tf.gather

Gather slices from params axis axis according to indices.

tf.gather(
    params,
    indices,
    validate_indices=None,
    axis=None,
    batch_dims=0,
    name=None
)

Gather slices from params according to indices. indices must be an integer
tensor of any dimension (usually 0-D or 1-D).

For 0-D (scalar) indices:
  output[i_0, ..., i_{axis-1}, i_{axis+1}, ..., i_n] =
    params[i_0, ..., indices, ..., i_n]

For 1-D (vector) indices:
  output[i_0, ..., i_{axis-1}, j, i_{axis+1}, ..., i_n] =
    params[i_0, ..., indices[j], ..., i_n]

The params tensor can have any rank >= 1. The indices tensor can have any rank.
The output rank = params.rank - 1 + indices.rank (when axis=0).

Args:
  params: The Tensor from which to gather values. Must be at least rank
    axis + 1. Can have any dtype.
  indices: The index Tensor. Must be one of the following types: int32, int64.
    The values must be in range [0, params.shape[axis]).
  validate_indices: Deprecated, does nothing. Indices are always validated on CPU.
  axis: A Tensor. Must be one of the following types: int32, int64. The axis
    in params to gather indices from. Defaults to the first dimension (axis=0).
    Supports negative indexing.
  batch_dims: An integer. The number of batch dimensions. Must be less than or
    equal to rank(indices). Defaults to 0.
  name: A name for the operation (optional).

Returns:
  A Tensor. Has the same type as params.

The values of indices must be in range [0, params.shape[axis]), otherwise the
behavior is undefined (may raise an error or return incorrect values).

Examples:
  params = tf.constant(['p0', 'p1', 'p2', 'p3', 'p4'])
  tf.gather(params, [3, 1, 2])  =>  ['p3', 'p1', 'p2']
  tf.gather(params, 2)          =>  'p2'

  params = tf.constant([[0, 1], [2, 3], [4, 5]])
  tf.gather(params, [0, 2])     =>  [[0, 1], [4, 5]]
