tf.argsort
Official URL: https://www.tensorflow.org/api_docs/python/tf/argsort
Signature: tf.argsort(values, axis=-1, direction='ASCENDING', stable=False, name=None)

Returns the indices that would sort a tensor along an axis.

Shape
Input:
- values: shape (d0, d1, ..., d(n-1))

Output:
- same shape as input: (d0, d1, ..., d(n-1))
- result entries are integer indices along the selected axis
