tf.nn.log_softmax
Official URL: https://www.tensorflow.org/api_docs/python/tf/nn/log_softmax
Signature: tf.nn.log_softmax(logits, axis=None, name=None)

Computes log-softmax activations along an axis.

Shape
Input:
- logits: shape (d0, d1, ..., d(n-1))

Output:
- same shape as logits: (d0, d1, ..., d(n-1))

Notes:
- often used when the class axis is the last dimension, such as (N, C) or (N, T, C)
