tf.boolean_mask
Official URL: https://www.tensorflow.org/api_docs/python/tf/boolean_mask
Signature: tf.boolean_mask(tensor, mask, axis=None, name='boolean_mask')

Applies a boolean mask to a tensor.

Shape
Input:
- tensor: shape (d0, d1, ..., d(n-1))
- mask: shape (d_axis, d_axis+1, ..., d_axis+k-1) or a prefix-compatible shape

Output:
- rank usually decreases or becomes partially data-dependent
- common example: tensor (N, D), mask (N,) -> output (M, D), where M depends on mask values

Notes:
- output shape depends on the number of True entries in the mask
- this API is useful but less statically predictable than reshape/transpose/concat
