tf.nn.conv1d
Official URL: https://www.tensorflow.org/api_docs/python/tf/nn/conv1d
Signature: tf.nn.conv1d(input, filters, stride, padding, data_format='NWC', dilations=None, name=None)

Computes a 1-D convolution.

Shape
Input:
- input: shape (N, W, C_in) for NWC or (N, C_in, W) for NCW
- filters: shape (K, C_in, C_out)

Output:
- shape (N, W_out, C_out) for NWC or (N, C_out, W_out) for NCW

Constraints:
- filter input channels C_in must match input channels
- W_out depends on stride, padding, dilation, and kernel size K
