tf.norm
Official URL: https://www.tensorflow.org/api_docs/python/tf/norm
Signature: tf.norm(tensor, ord='euclidean', axis=None, keepdims=None, name=None)

Computes vector norms, matrix norms, and tensor norms.

Shape
Input:
- tensor: shape (d0, d1, ..., d(n-1))

Output:
- if axis is None and keepdims=False: scalar ()
- if axis is an integer: rank decreases by 1 unless keepdims=True
- if axis is a 2-tuple for matrix norms: two dimensions are reduced
- examples:
  - (N, D), axis=1 -> (N,)
  - (B, M, N), axis=(-2, -1) -> (B,)
  - keepdims=True retains size-1 dimensions
