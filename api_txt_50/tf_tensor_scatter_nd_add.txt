tf.tensor_scatter_nd_add
Official URL: https://www.tensorflow.org/api_docs/python/tf/tensor_scatter_nd_add
Signature: tf.tensor_scatter_nd_add(tensor, indices, updates, name=None)

Adds sparse updates into an existing tensor and returns the result.

Shape
Input:
- tensor: shape (s0, s1, ..., s(r-1))
- indices: shape (i0, i1, ..., i(m-1), k)
- updates: shape indices.shape[:-1] + tensor.shape[k:]

Output:
- shape identical to tensor: (s0, s1, ..., s(r-1))
