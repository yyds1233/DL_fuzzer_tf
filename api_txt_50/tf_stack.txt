tf.stack
Official URL: https://www.tensorflow.org/api_docs/python/tf/stack
Signature: tf.stack(values, axis=0, name='stack')

Stacks tensors along a new axis.

Shape
Input:
- values: a list of tensors with identical shape (d0, d1, ..., d(n-1))

Output:
- shape with one new axis of length len(values)
- examples:
  - stack 4 tensors of shape (H, W) at axis 0 -> (4, H, W)
  - stack 4 tensors of shape (N, C) at axis 1 -> (N, 4, C)

Constraints:
- every input tensor must have exactly the same shape
- output rank is input rank + 1
