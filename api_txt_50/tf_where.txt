tf.where
Official URL: https://www.tensorflow.org/api_docs/python/tf/where
Signature: tf.where(condition, x=None, y=None, name=None)

Returns indices where condition is true, or selects elements from x and y.

Shape
Case 1: condition only
- condition: shape (d0, d1, ..., d(n-1))
- output: shape (M, n), where M is the number of True elements

Case 2: condition, x, y
- condition, x, y: broadcast-compatible shapes
- output: broadcasted shape (...)
- examples: condition (N, C), x (N, C), y (N, C) -> output (N, C)

Notes:
- the index-returning form is data-dependent
- the select form has static broadcast-based shape rules
