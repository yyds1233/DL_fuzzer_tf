tf.concat
Official URL: https://www.tensorflow.org/api_docs/python/tf/concat
Signature: tf.concat(values, axis, name='concat')

Concatenates tensors along an existing axis.

Shape
Input:
- values: a list of tensors with the same rank r
- each tensor may differ only on the concatenation axis
- common example: (N, H, W, C1) and (N, H, W, C2)

Output:
- same rank r
- output shape matches inputs on all non-concat axes
- concat result example: (N, H, W, C1 + C2)

Constraints:
- all non-concat dimensions must match exactly
