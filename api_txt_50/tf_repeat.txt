tf.repeat
Official URL: https://www.tensorflow.org/api_docs/python/tf/repeat
Signature: tf.repeat(input, repeats, axis=None, name=None)

Repeats elements of a tensor along an axis.

Shape
Input:
- input: shape (d0, d1, ..., d(n-1))

Output:
- if axis is None, input is flattened first and output is 1-D
- if axis is set, output rank equals input rank
- examples:
  - repeat (N, C) along axis 1 by k -> (N, C * k)
  - repeat (N,) by k -> (N * k,)

Notes:
- exact output length may depend on repeats if repeats is a vector
