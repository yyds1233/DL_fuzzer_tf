tf.nn.softmax
Official URL: https://www.tensorflow.org/api_docs/python/tf/nn/softmax
Signature: tf.nn.softmax(logits, axis=None, name=None)

Computes softmax activations along an axis.

Shape
Input:
- logits: shape (d0, d1, ..., d(n-1))

Output:
- same shape as logits: (d0, d1, ..., d(n-1))

Notes:
- commonly used on class dimension, for example (N, C) -> (N, C) or (N, H, W, C) -> (N, H, W, C)
