tf.gather_nd
Official URL: https://www.tensorflow.org/api_docs/python/tf/gather_nd
Signature: tf.gather_nd(params, indices, batch_dims=0, name=None)

Gathers slices or elements from params using n-dimensional indices.

Shape
Input:
- params: shape (d0, d1, ..., d(n-1))
- indices: shape (i0, i1, ..., i(m-1), k)

Output:
- if k == n and batch_dims == 0, output shape is (i0, i1, ..., i(m-1))
- in general, output shape is indices.shape[:-1] + params.shape[k:]
- examples:
  - params (B, H, W, C), indices (N, 3) -> (N, C) if k=3
  - params (M, K), indices (P, 2) -> (P,)

Constraints:
- last dimension k of indices selects how many leading dimensions of params are indexed
