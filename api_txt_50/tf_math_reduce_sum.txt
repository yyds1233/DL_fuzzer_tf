tf.math.reduce_sum
Official URL: https://www.tensorflow.org/api_docs/python/tf/math/reduce_sum
Signature: tf.math.reduce_sum(input_tensor, axis=None, keepdims=False, name=None)

Reduces a tensor by summing elements across dimensions.

Shape
Input:
- input_tensor: shape (d0, d1, ..., d(n-1))

Output:
- if axis is None and keepdims=False: scalar ()
- if axis is a single dimension with keepdims=False: rank decreases by 1
- if keepdims=True: reduced axes are kept as size 1
- examples:
  - (N, C, H, W), axis=1 -> (N, H, W)
  - (N, C, H, W), axis=(2, 3), keepdims=True -> (N, C, 1, 1)
